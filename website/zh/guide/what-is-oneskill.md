# 什么是 OneSkill?

你的 AI Agent 很强，但它还可以更强。

**OneSkill** 是连接你的 Agent 与 [OpenSkills](https://github.com/Starttoaster/openskills) 庞大生态的桥梁（但使用繁琐，还需要寻找合适的资源）。只需安装这一个工具，你的 Agent 就能获得自我进化的能力：**发现新skill、更方便进行安装新技能、支持codex、gemini等众多工具**。

不再需要手动下载文件、手动修改配置。一切都自动化了。

## 核心价值

*   **海量能力库**: 瞬间接入社区维护的数千个 Skill（Web浏览、数据库管理、代码审计、云设施运维等）。
*   **一键式安装**: 一个命令，完成下载、依赖处理和环境配置。
*   **Gemini 完美适配**: 开发对 Gemini CLI 的 `map` 功能，自动将openskills下载的 Skill 注入到 Gemini 的路径中。
